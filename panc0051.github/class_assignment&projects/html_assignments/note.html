Paragraph Tag
The paragraph tag describes a text paragraph. It is an aspect of the block-level
and often begins on a new line. Browsers will automatically add margins before and after
each paragraph.
Body Tag
The body element represents the main content of the document. It typically wraps around
all of the content that will be displayed on screen, such as headings, paragraphs, hyperlinks,
images, forms, tables, lists, videos and so on.
Article Tag
A section of content that forms an independent part of a text, such as a blog post, article, or other
self-contained unit of material, represents the article element.
Blockquote Tag
The blockquote tag shows that an extended quotation is an enclosed text.
Cite Tag
A citation or reference to another source is indicated by the Cite tag.
