What exactly is Bootstrap?

Bootstrap is a free, fast, and easier front-end web development framework Bootstrap contains HTML and CSS design templates for typography, forms, buttons, tables, browsing models, image carusel and many more. You can also create responsive designs easily by Bootstrap.

It uses utility classes for typography and spacing to space content out within the larger container.

Split button

In the same way, split button dropdowns are created with practically the same markdown as single button drops but added. For proper spacing around the dropdown caret, dropdown-toggle-split.This supplementary class is used to reduce the horizontal padding by 25 percent on either side of the care and remove the left margin for regular button downdrops. These additional changes keep the attention focused on the split button and give a more suitable hit area next to the main button.

Multiple targets
